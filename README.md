# DER---DWES-Proyecto-1Trimestre---Recetas
Proyecto primer trimestre Dana Espinosa Ramos DWES

-- Recetas de cocina --
Mi Proyecto consiste en una web en PHP usando Twig y MySQL para gestionar recetas de cocina. Permite a los usuarios registrarse, iniciar sesión, añadir, editar y eliminar recetas, así como ver recetas generales (es decir un listado) y filtrarlas por país de origen. Utilizando una interfaz con colores pastel y agradable a la vista, así como una barra de navegación con enlaces a perfil, recetas y con el logout
